pyspec.file.spec module¶

filespec¶

Description¶

This module offers an interface to files with data in the spec file format. The format specificiation can be consulted online at the certif.com website.

A spec file normally consists of a series of header blocks and scan blocks. Data comes after scan blocks. Sometimes comment lines can be also be found in between blocks. These comment lines could contain for example the result of user calculations after a scan, or pure comments. For practical purposes comment lines will be considered to belong to the preceding scan or header block.
